# Calculadora de propina-Windows Forms
Diseñé una calculadora de propinas donde el usuario ingresa el monto total de su factura y el porcentaje que desea dejar de propina. Por ejemplo, si la cuenta es de $200 y el usuario decide dejar un 50% de propina, el programa calcula que la propina son $100 y muestra un total de $300. Esto ayuda a las personas a calcular rápidamente cuánto deben pagar sin tener que hacer reglas de tres mentalmente.

Para lograr esto, en mi aplicación utilicé los siguientes controles:

3 Labels (Etiquetas): Sirven para dar instrucciones claras al usuario, indicando dónde colocar la cuenta, el porcentaje y dónde aparece el resultado.

2 TextBox (Cuadros de texto): Son los espacios interactivos donde el usuario escribe el monto de la factura y el porcentaje de propina que desea aplicar.

1 Button (Botón): Es el control encargado de procesar la operación matemática al hacer clic, activando el cálculo del porcentaje y mostrando el total final.
